Lesson Connection

The sport of lacrosse has created so many opportunities for me so I wanted to build an app to help youth and inexperienced players improve their game. Lesson Connection facilitates the connection between gain access to private lessons from some of the best lacrosse players in the country.

NYC Farmers' Markets

This project utilizes a D3 (Data-Driven Documents) map that shows the locations of farmers' markets in the New York City area in 2013. My group and I implemented a jQuery UI slider to display circles on the map indicating farmers' markets that are open for service on that particular day.

DevelUp

This application provides developers with tools to help improve their productivity. Developers can use information from the application to discover new locations to code, improve Git commit frequency with commit reminders, assess and track productivity against a host of different parameters, and connect with other developers in the area.
